Installation

  • New install
  • Upgrade

  • File List

New install

  1. If you have any versions prior to v1.5.6c installed, please check the upgrade tab for instructions
  2. Un-zip the package to a folder on your computer
  3. Rename the folder "YOUR_ADMIN" to your actual admin folder name.
  4. Copy all files & folders to your server. Below is a list of the files.

Upgrade

Use the list of files in the installation tab for comparison with your current installation, to see if any files have been added, or removed formthe package.
The easiest way is to use a file comparison program like Beyond Compare, or Winmerge.

Un-install

  • Go to "Localization => Languages"
  • Select "Nederlands" (Dutch) and press delete
  • Delete all the files from your server. The fastest way is to delete:
    • includes/languages/dutch.php
    • includes/languages/YOUR_TEMPLATE/dutch.php (if present)
    • includes/languages/dutch (entire folder)
    • YOUR_ADMIN/includes/languages/dutch.php
    • YOUR_ADMIN/includes/languages/dutch (entire folder)
  • Run the uninstall.sql in "Tools => Install SQL Patches" of your Zen Cart admin panel

Activate the laguage pack in your admin

Log in in uw Admin en ga naar Localization => Languages klik op de "new language" knop.

  1. set "Name:" to "Nederlands"
  2. set "Code:" to "nl"
  3. set "Image:" to "icon.gif"
  4. set "Directory:" to "dutch"
  5. set "Sort Order:" with a number you choose
  6. Set as default: Check this box if you want this language set as deafualt
  7. click on thee "insert" button

Making changes

When making changes to the files in this language pack, save the changed files in the override folder. For example: includes/languages/dutch.php should be saved in includes/languages/YOUR_TEMPLATE/dutch.php, Where YOUR_TEMPLATE substitutes for your template name. You should not make any changes to the original (core) files, because this will make future upgrades more difficult. For more information on how to use the override system, please see this tutorial.

Add a currency to the language

Als u wilt dat uw klanten automatisch veranderen van valuta naar Euro als ze voor de Nederlandse taal kiezen in uw winkel, gan dan in de Admin naar Configuration => My Store en verander 'Switch To Default Language Currency' naar true. Dan wordt de valuta automatisch aangepast als er van taal veranderd wordt.

Als u problemen ondervind met deze add-on module, bekijk dan eerst de Dutch language pack support thread, om te kijken of uw vraag al beantwoord is. Als dat niet het geval is kunt u een vraag plaatsen. Alle support gaat via het Zen Cart support forum zodat alle gebruikers van Zen Cart er van kunnen profiteren.
