Installation

  1. If you have any versions prior to v1.5.6b installed, please check the upgrade tab for instructions
  2. Un-zip the package to a folder on your computer
  3. Rename the folder "YOUR_ADMIN" to your actual admin folder name.
  4. Copy all files & folders to your server. Below is a list of the files.

Upgrade

Use the list of files in the installation tab for comparison with your current installation, to see if any files have been added, or removed formthe package.
The easiest way is to use a file comparison program like Beyond Compare, or Winmerge.

Un-Install

  • Go to "Localization => Languages"
  • Select "Nederlands" (Dutch) and press delete
  • Delete all the files from your server. The fastest way is to delete:
    • includes/languages/dutch.php
    • includes/languages/YOUR_TEMPLATE/dutch.php (if present)
    • includes/languages/dutch (entire folder)
    • YOUR_ADMIN/includes/languages/dutch.php
    • YOUR_ADMIN/includes/languages/dutch (entire folder)
  • Run the uninstall.sql in "Tools => Install SQL Patches" of your Zen Cart admin panel

Activate this Language Pack in your Admin

Sign in to your Admin, and go to "Localization => Languages" click on "New Language".

  1. Set "Name:" to "Nederlands"
  2. Set "Code:" to "nl"
  3. Set "Image:" to "icon.gif"
  4. Set "Directory:" to "dutch"
  5. Set "Sort Order:" to a value of your choice
  6. Set as default: vink deze box aan als u deze taal als standaard wilt instellen - anders kunt u het vinkje leeg laten
  7. Click on "Insert"

Configuration Options

If you plan to make any alterations to the defines in this language pack, be sure to save the altered file in the template override folder. For example: includes/languages/dutch.php becomes includes/languages/YOUR_TEMPLATE/dutch.php. Where YOUR_TEMPLATE is substituted with the actual name of your template folder name. It is not a good idea to save the default files, as this will complicate future updates. For more information on the override system, please read this tutorial.

Apply currencyto this language pack

If you wish to automatically change the currency to Euros when a customer selects the Dutch language, go to "Configuration => My Store" and change "Switch To Default Language Currency" to true.

Als u problemen ondervind met deze add-on module, bekijk dan eerst de Dutch language pack support thread, om te kijken of uw vraag al beantwoord is. Als dat niet het geval is kunt u een vraag plaatsen. Alle support gaat via het Zen Cart support forum zodat alle gebruikers van Zen Cart er van kunnen profiteren.
