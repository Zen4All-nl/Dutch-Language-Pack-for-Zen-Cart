Readme File for Dutch Language Pack v1.5.3
for Zen Cart Version 1.5.3

The purpose of this mod is to provide a Dutch Language Pack to work with Zen Cart Version 1.5.3 .

History and Change Log

2014-11-18
  • Changes to the files, aaded numerous new defines, and deleted obsolete ones.
  • Dit pack only provedis UTF-8

2012-01-17 v1.5.1:
  • This is the firts official release for Zen Cart 1.5
  • This package is build from the version for Zen Cart 1.39h, supplemented with data from multiple forum members.
  • This pack now provides the choice of either utf-8 or iso-8859-1

Installation

  1. Install the files

    1. Unzip the release package to a new folder.
    2. Rename the admin folder to reflect the name of your admin folder
    3. Drag all folders into the root folder for your site (that's the one with an includes folder already as well as images, cache, media and a number of others).
    4. Upload all files of this package to your live server.
    5. If you are upgrading from Dutch Language Pack 1.39h delete the files:
      • /admin/includes/languages/dutch/cache.php
      • /admin/includes/languages/dutch/email_welcome.php
      • /admin/includes/languages/dutch/extra_definitions/editors_list.php (emptied - file can be deleted)
      • /admin/includes/languages/dutch/extra_definitions/onlyFor139.php (emptied - file can be deleted)
      • /includes/languages/english/tell_a_friend.php
      • /includes/languages/english/modules/shipping/ups.php
      • /includes/languages/english/modules/shipping/usps.php

  2. Activate this language pack in your admin

    Login to your Admin and in Localization => Languages click the "new language" button.

    1. set "Name:" to "Dutch"
    2. set "Code:" to "nl"
    3. set "Image:" to "icon.gif"
    4. set "Directory:" to "dutch"
    5. set "Sort Order:" to your choice of order
    6. Set as default: check this box if you want this language to be your store default language - otherwise leave blank
    7. click the "insert" button

  3. Making changes

    If you intend to make changes to any of the defines contained in the files of this package, save the changed files to the relevant template folder of your install. It is not recommended to overwrite the existing core files as it will make future upgrades much more difficult. For information about how to use the override system please consult this tutorial.

  4. Assign currency to this language pack

    If you want your customers to automatically switch to the Dutch default currency Euro when they select the German language at your store front, then go to Admin => Configuration => My Store and set 'Switch To Default Language Currency' to true. This will automatically switch to the language's currency when it is changed
