Installation

  1. If you have any versions prior to v1.5.6b installed, please check the upgrade tab for instructions
  2. Un-zip the package to a folder on your computer
  3. Rename the folder "YOUR_ADMIN" to your actual admin folder name.
  4. Copy all files & folders to your server. Below is a list of the files.

Un-Install

  1. Suggestion to save time...
    First, make sure all your products are completely setup. Especially the product options. Whenever you make an edit on a child product of a bundle, you should review your bundle settings.
  2. When creating a new product, choose 'Product - Bundle' from the drop-down menu next to the New Product button.
  3. Setup the product details, like any other product. Name, description, etc
  4. At the bottom of the page you can start adding products to your bundle.
  5. Click on the product name after adding to set any of the options.
  6. Preview, then Insert/Update.
  7. Done!

Information about the setting within a bundled product are available within the module.

Background information

This all started in 2014 after purchasing a copy of the original v1.2 module

This Module is the ZenBundles module updated for Zen Cart v1.5.6. ZenBundles is a module that allows you to bundle individual product in your catalog into one product.

The new 2.0.0 version here is our attempt to update the module to work with Zen Cart v1.5.6. I believe that the Zen Cart v1.5.6 changes we made are are correct.

  • This new module was created because unfortunately not only did the module's original author fail to update the ZenBundles package to work with Zen Cart v1.5.x (though he says his module is available in a v1.5.x version), he has consistently responded to ALL of our requests to update the ZenBundles module code that is clearly from the v1.3.9 version of Zen Cart.
  • And in case anyone is wondering, we have tried very hard to get his cooperation in this effort to no avail.
